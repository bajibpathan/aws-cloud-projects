# 📸 Event-Driven Image Processing

## Overview

This project demonstrates how to build an event-driven image processing pipeline using AWS.

The goal is to automatically process uploaded images without manual intervention by integrating Amazon S3, AWS Lambda, DynamoDB, and CloudWatch.

This project is being built incrementally to understand each AWS service and how they work together.
