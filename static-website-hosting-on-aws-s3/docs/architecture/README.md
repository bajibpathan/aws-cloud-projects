# Architecture Overview

This project uses a very simple AWS architecture designed for beginners learning cloud fundamentals.

## 🌐 How the System Works

The website is hosted using this basic flow:

![Architecture Diagram](../diagrams//high-level-architecture.png)

- **Amazon S3** stores and serves the static files (HTML, CSS, JS).
- The S3 bucket is configured for **static website hosting**.
- A public website endpoint is generated by S3.
- Users access the site directly through that endpoint.

This is one of the simplest ways to host a website on AWS and is perfect for learning.

## 📄 Why This Architecture

- No servers to manage  
- Very low cost  
- Easy to deploy  
- Great for beginners  
- Helps build foundational AWS knowledge

As I learn more, I may add CloudFront, Route 53, or Lambda in future versions.
