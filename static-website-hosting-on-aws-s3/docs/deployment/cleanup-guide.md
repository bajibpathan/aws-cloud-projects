# 🧼 Clean‑up Instructions

If you no longer want this project hosted on AWS, you can safely remove all resources to avoid any unexpected charges.  
These steps fully delete your S3 static website setup.

---

## 🗑 Step 1: Delete All Objects Inside the S3 Bucket

1. Open the **Amazon S3** console.
2. Click on your bucket name.
3. Go to the **Objects** tab.
4. Select **all files**.
5. Click **Delete**.
6. Type **permanently delete** to confirm.

📌 *Important:*  
An S3 bucket **must be empty** before AWS allows you to delete the bucket itself.

---

## 🗑 Step 2: Delete the S3 Bucket

1. Return to the main **S3 bucket list** page.
2. Select your now‑empty bucket.
3. Click **Delete**.
4. Type the **bucket name** to confirm deletion.

Once this is done, all resources for this project are removed from your AWS account.

---

## ✔️ Clean‑up Complete

Your static website hosting setup has been fully removed.  
This ensures you will not incur any future charges related to this project.
